# 🚀 TaskPilot AI

TaskPilot AI is an AI-powered productivity assistant built using Flask and Google Gemini AI. It helps users plan tasks intelligently based on task name, deadline, and priority.

## ✨ Features

- 🤖 AI-powered task analysis
- 📅 Smart planning based on deadline
- ⚡ Priority-based suggestions
- 📈 Difficulty estimation
- 💡 Motivational tips
- 🎨 Simple and user-friendly interface

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- Google Gemini AI

## ▶️ How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python app.py
```

4. Open your browser

```
http://127.0.0.1:5000
```
